  • What if we are not satisfied with your service?

    If you are not satisfied with our service, no questions asked and we will refund the whole payment straight away. The refund should be asked before the project files are transferred.

  • What is your turnaround times?

    We try our best to finish the project within 24 hrs if the design file is a layered and is one page. If the design files are not layered and there are multiple pages additional time is required.   • How does the process work?

    Step 1 :-
    The client submits the PSD to HTML/XHTML/CSS project using the order form along with all the design files and fonts. In return he gets a confirmation mail with the project area access. The status of the project will be “new” at this stage.

    Step 2 :-
    HTMLSliceMate.com project manager gets the order, check all relevant details and pass it on to the HTMLSliceMate.com markup coder. Any further information needed for the project is been asked in the project area discussion board . The status of the project will be “working” at this stage.

    Step 3 :-
    After markup coding, the project is passed on to the HTMLSliceMate.com QA team. HTMLSliceMate.com QA team tests the codes on all major browsers and operating systems for compatibility.

    Step 4 :-
    HTMLSliceMate.com QA team then uploads the completed project onto the project area and asks the client to review and provide feedback in the project area discussion board.

    Step 5 :-
    HTMLSliceMate.com markup coder checks the feedback, make appropriate changes and transfer the files to the QA team. HTMLSliceMate.com QA team does final checking and upload the finished files for the client to download. The status of the project will be “completed” at this stage.

However, we do provide invoice to you for any transaction you made to us on paypal or in our bank using wire transfer if you request us.

    For Design to HTML / HTML5 / Email / Mobile services, we take 100% money upfront and for CMS (WordPress, Joomla!, Drupal and Magento) projects, we take 50% of the total payment at the start of the project and 50% at the end of the project.   • What are the payment methods?

    Payments are done through credit cards or paypal using fully secured paypal payment gateway.

  • What are your pricings?

    The basic single page package starts from 45 USD and for detailed pricing, please go to the order form. We offer 50% reduction on the additional pages.

  • Can I choose you to design my site and use my own web hosting?

    Currently we are not taking any design jobs. But we are coming up with a new design service website for design jobs shortly.

  • What types of design formats do you accept?

    We accept almost all types of design file formats including Adobe Photoshop (.psd), Adobe Illustrator (.ai), Adobe InDesign (.indd), Adobe Fireworks (.png), PDF, EPS, JPEG, GIF and TIFF. However, layered design files are the best option for a speedy delivery. Fonts other than web fonts need to be submitted along with the design files. Width need to be specified when submitting the design files in PDF.

  • What is W3C valid Markup/CSS?

    When a site meets W3C standards, it is like an approval of the main web guidelines, specifications, and tools, including HTML and CSS. Our work is always 100% validated HTML/XHTML & CSS.

  • Which CSS frameworks are supported?

    We support all major CSS frameworks like Twitter Bootstrap, 960 Grid System, Gumby, Blueprint, Zurb Foundation, Boilerplate, YAML etc. Please specify the required framework in the order options or in the project notes and the choice is completely free of cost. We use LESS and SASS upon request and there will be additional charges per page.

  • What browsers do you support?

    Our code is compatible with all modern browsers like IE8+, Opera, Safari, Firefox and Google Chrome. If you require IE7 compatibility, it can be selected as a separate option at the time you place your order.
